# Shortcutkeys

A simple direct pin macropad for a 6 shortcuts : copy, paste, new tab, close tab, undo, redo.

## BOM
1 unsoldered Seeed XIAO RP2040

6x MX-Style switches

6x white blank DSA keycaps

4x M3x16mm screws

4x M3x5mx4mm heatset inserts
